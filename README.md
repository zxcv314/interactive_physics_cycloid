# Cycloid Grand Lab

An interactive web simulator exploring the mathematical beauty and physics principles of the cycloid curve.

- 주요 기능 (Key Features)

1. 사이클로이드
* 천장에 붙은 원이 구를 때 원 위의 한 점이 그리는 사이클로이드 곡선의 기하학적 정의를 실시간으로 보여줍니다.
* 스크롤 인터랙션을 통해 원이 구르는 모습을 동적으로 제어하며, 매개변수 방정식과 실제 궤적을 직관적으로 매칭할 수 있습니다.

2. 최단강하곡선 (Brachistochrone)
* 두 점 사이를 이동할 때 직선이나 다른 곡선보다 사이클로이드 궤적이 왜 가장 빠른지 눈으로 증명합니다.
* 마우스로 직접 화면에 가이드 라인을 그려보며 이론상의 곡선과 실시간 낙하 속도 대결을 펼칠 수 있습니다.

3. 등시성 (Tautochrone)
* 곡선의 어느 위치에서 공을 떨어뜨려도 최하점까지 도달하는 시간이 완벽하게 같다는 물리 법칙을 실험합니다.
* 사용자가 원하는 높이에 여러 개의 공을 직접 배치하고 중력을 켜서 동시에 도달하는 시각적 경험을 체험할 수 있습니다.

4. Art of Math (Mathematical Art)
* 사이클로이드의 응용 형태인 하이포사이클로이드(Spirograph)를 활용하여 기하학적 문양을 디자인하는 캔버스입니다.
* 외경, 내경, 점의 거리 슬라이더를 조작함에 따라 수만 개의 좌표가 실시간으로 재연산되어 유기적인 예술 작품을 탄생시킵니다.

- 기술 스택 (Tech Stack)
* HTML5 Canvas API
* Tailwind CSS
* Vanilla JavaScript (ES6+)
